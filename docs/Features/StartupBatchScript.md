The batch file can be created and modified to your likings at `I:\Apps\SalsaNOW\StartupBatch.bat`, whenever SalsaNOW reaches the batch startup part it starts the batch from `I:\Apps\SalsaNOW\StartupBatch.bat` if one exists, this can be used as windows startup to start anything you want whenever you start a GeForce NOW Session with SalsaNOW.

Leveraging bing photo of the day API we can now pull photos/images of the day from bing and apply them as wallpaper on WinXShell, photos reset everyday at 6:00 PM GMT+11 Canberra Australia.<br>
In order to apply the feature you must modify the config file at `I:\Apps\SalsaNOW\SalsaNOWConfig.ini` and modify the value from `BingPhotoOfTheDayWallpaper = "0"` to `"1"`, if config line `BingPhotoOfTheDayWallpaper = "0"` does not exist, you must add it manually anywhere in the file !

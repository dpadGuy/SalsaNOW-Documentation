SalsaNOW supports a special way of installing .NET 5.0 up to 10 via a special executable that installs the versions at path ``C:\Users\kiosk\AppData\Local\Microsoft\dotnet``

In order to run the .NET special installer you must run the following command inside CMD:
``"I:\Apps\SalsaNOW\SilentApps\InstallNet.exe"``
